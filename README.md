# 天使在线 · 华学苑官网重搭

本仓库用于基于旧站内容素材，重搭「天使在线 · 华学苑」官网，并通过 GitHub Pages 做生产部署。项目优先保证中文内容、移动端体验、视觉完成度和咨询转化路径。

## 页面结构

- `/`：官网首页，承接品牌认知、项目入口、信任证明和咨询转化。
- `/summer-hua/`：秦岭成长夏令营页面，突出营期信息、适合对象、流程和申请入口。
- `/life/`：生命成长与书籍共读页面，展示长期成长陪伴、书籍共读和真实场景。
- `/consult/`：预约咨询页面，在静态托管上整理并复制咨询信息，用户通过微信或电话发送。
- `/portal/`：家长与学员入口页。
- `/camps/`：营地项目列表页。
- `/articles/`：文章与内容入口页。

## 本地预览

```bash
python3 -m http.server 4173
```

打开 `http://127.0.0.1:4173/` 即可预览。

## GitHub Pages 部署

本项目是纯静态站点，不需要构建步骤：

- Pages source：`main` 分支根目录。
- 生产地址：`https://jasoneip01-pixel.github.io/angel/`
- `.nojekyll` 保证 GitHub Pages 按静态文件原样发布。
- 咨询入口在 GitHub Pages 上不依赖服务端表单，前端会整理并复制咨询信息，用户通过微信或电话发送。
